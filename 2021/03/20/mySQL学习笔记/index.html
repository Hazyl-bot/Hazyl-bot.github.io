高级查询：子句和正则表达式

子句 参数和作用
LIMIT 两个参数x,y，表示从x行开始查询共y行数据，如果只填一个参数，则默认为0，y
ORDER BY 一个字段名参数，表示按照这个字段排序，后加DESC可表示降序排列

模糊查询

​ 很多时候我们不知道需要文档的全名，好在Mysql支持模糊查询。

​ SELECT * FROM `user` WHERE nickname LIKE "xiao%" 可以帮助我们选择昵称以xiao开头的所有用户。这里%表示任意个数的任意字符，可以加在前面和后面。对应的，_表示单个任意字符。
