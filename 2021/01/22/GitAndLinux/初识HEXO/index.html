初识HEXO

hexo是一个博客框架，可以帮助我们把md文档通过几条简短的命令变成好看的博客网页

点击这里本地安装

基本命令

创建博客文件夹 hexo init
安装发布工具 npm install hexo-deployer-git –save
清除缓存，用于快速发布博客 hexo clean
生成新的静态文件 hexo generate
把生成的文件部署到博客上 hexo deploy
创建一个博客文件（.md) hexo new blogName
